# Madlibs-Generator
A fun Python script that prompts the user for nouns, verbs, adjectives, and other word types, then inserts them into a story template to generate a personalized and often humorous Madlibs-style result.

## Features

Collects multiple user inputs

Inserts words into a formatted story template

Generates different stories with the same template

Simple, clean code for beginners

## What I Learned

String formatting (f-strings or .format())

Organizing user input logically

Working with templates and text output

Basic function structure

## How to Run
have file "madlib.txt" made with madlib sentence format to input into code as file

python main.py
